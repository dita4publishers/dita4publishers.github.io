epub.copy.system.css Ant parameter

Indicates whether or not to copy the built-in CSS files to the EPUB.

When set to "false" turns off copying of the built-in CSS files to the EPUB. The default is "true" (copy the system CSS files).

Value

One of the keywords "true" or "false".

Default

"true".
