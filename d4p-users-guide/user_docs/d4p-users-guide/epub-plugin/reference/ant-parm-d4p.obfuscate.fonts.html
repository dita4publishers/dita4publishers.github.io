epub.obfuscate.fonts Ant parameter

Indicates whether or not to obfuscate fonts that are marked as obfuscated in the font manifest used when generating the EPUB.

See Embedding Fonts in Your EPUB and epub.font.manifest Ant parameter.

Value

true
Fonts marked for obfuscation will be obfuscated when stored in the generated EPUB package.
false
No fonts will be obfuscated.

Default

"false".
