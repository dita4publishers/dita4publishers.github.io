Requirements

The HTML5 plugin requires:
  • An installation of DITA OT versions 1.6.3, 1.7.5 or 1.8.4;
  • Ant 1.8.4 or superior;
  • Java 1.6 or superior.
The HTML5 plugin has been tested sucessfully using a fresh installation on:
  • CentOS 6, ant 1.8.4, java 1.6.0_20, OpenJDK Runtime Environment (IcedTea6 1.9.8) (rhel-1.22.1.9.8.el5_6-x86_64);
  • Mac OS X.8.5, ant 1.9.4, java 1.6.0_65 on DITA_OT versions 1.6.3, 1.7.5 and 1.8.4;
  • Windows 7 ant 1.9.4, java 1.6.0_65 on DITA_OT versions 1.6.3, 1.7.5 and 1.8.4.

However, the plugin is mostly developped on Linux technology, so any information that might improve the plugin on Windows will be appreciated.
