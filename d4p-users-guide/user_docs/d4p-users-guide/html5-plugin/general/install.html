Specific installation notes for developers

If you want to do development with HTML5 plugin, you will need to install the YUI compressor tool.

  1. Download the YUI compressor version 2.4.7
    Use the 2.4.7 version, the version 2.4.8 does not work properly on Windows due to a path resolution issue.
  2. Extract the content into the org.dita4publishers.html5 plugin directory
